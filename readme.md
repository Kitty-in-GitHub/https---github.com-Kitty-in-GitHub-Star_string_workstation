# 星弦的社团工作随机自动分配机
*工作、生活，一切是那么美妙！*

自动、高效、迅速的为浙江大学学生星弦科幻协会的成员分配工作！

## 文件说明：
- format_read.py：实现功能的脚本
- 1号表干员信息表.csv：记录干员信息的表格，记录信息包括qq号、群昵称、手机号、已从事过的工作数量、倾向于从事的工作类型
- 2号表工作流记录表.csv：
- 3号待完成工作表.csv：

三张csv表格可以手动创建，也可以通过脚本自动创建。其中，1号表干员信息表.csv需要您手动填写，其他表格可以通过程序填写

## 使用说明：

把3个csv文件和py脚本放在同一文件夹下，控制台进入这个文件夹，然后跑py脚本即可

为了星弦！
